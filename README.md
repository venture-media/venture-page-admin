## Description

Adds a "No index" checkbox column to the Pages admin list table and a simple 301 redirect manager dashboard widget.

## Features

- "No index" column with AJAX checkbox on Pages list table
- Outputs `<meta name="robots" content="noindex">` on selected pages
- Dashboard widget to add and manage 301 redirects

## Installation

1. Upload the `venture-page-admin` folder to `/wp-content/plugins/`
2. Activate **Venture Page Admin** in WordPress

## Usage

### No Index
- Go to **Pages → All Pages**
- Use the "No index" checkbox column

### Redirects
- Go to the WordPress Dashboard
- Use the **Venture Redirects** widget to add or delete redirects
